Esta carpeta es para guardar tus sprites/texturas.

Formatos recomendados:
- PNG (con transparencia)
- JPG (sin transparencia)

Tamaños recomendados para pixel art:
- 16x16
- 32x32
- 64x64

Configuración en Unity al importar:
1. Selecciona la imagen
2. Inspector:
   - Texture Type: Sprite (2D and UI)
   - Pixels Per Unit: 100 (o 16/32 para pixel art)
   - Filter Mode: Point (pixel art) o Bilinear (suave)
3. Click Apply
